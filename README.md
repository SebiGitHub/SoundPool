# SoundPool (Android)

## Qué es
Mini-proyecto para practicar efectos de sonido cortos en Android usando SoundPool (ideal para juegos/UI feedback).

## Stack
- Kotlin
- Android Studio
- API SoundPool
- Recursos de audio (mp3)

## Features
- Cargar sonidos y reproducirlos con botones/eventos
- Control de volumen, repetición y prioridad
- Liberación correcta de recursos (`release`)

## Cómo ejecutar
1. Clona el repo
2. Abre en Android Studio
3. Ejecuta y prueba los botones de sonido

## Qué aprendí
- Diferencia entre audio “corto” (SoundPool) y reproducción larga (MediaPlayer/ExoPlayer)
- Gestión de recursos y latencia
- Cómo estructurar un mini-motor de sonido reutilizable
